# MEMORY.md - Kai 的长期记忆

这里存储从经验中提炼的原则、每次对话都会用到的内容。

---

## Memory 文件使用规范

| 文件 | 存什么 | 例子 |
|------|--------|------|
| **USER.md** | 用户是谁 | 姓名、时区、偏好 |
| **TOOLS.md** | 环境配置 | ID、参数、设备信息 |
| **MEMORY.md** | 经验教训、原则 | 从实践中提炼的通用规律 |
| **memory/YYYY-MM-DD.md** | 每日活动流水 | 今天做了什么 |

## 工作原则

### 任务执行策略

- **耗时 >5 分钟的任务应该派生 subagent**，让用户不用等待
- 多步骤、有不确定性的任务适合派生 subagent
