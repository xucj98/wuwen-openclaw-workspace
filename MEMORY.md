## 工作原则

### 任务执行策略 (2026-03-30)

- **耗时 >5 分钟的任务应该派生 subagent**，让用户不用等待
- **多步骤、有不确定性的任务** 适合派生 subagent
- 单一可预测的命令可以自己执行，但如果涉及安装/下载等耗时操作，考虑 subagent

## 飞书权限体系

### 关键发现

1. **机器人无法直接添加为知识库成员**（飞书只支持添加用户或群）
   - 解决方案：创建群 → 机器人加入群 → 群添加为知识库管理员

2. **创建知识库需要用户身份**
   - `tenant_access_token`（应用身份）无法创建知识库
   - 需要 `user_access_token`（用户身份）通过 OAuth 授权

3. **Wiki API 与 Docx API 的关系**
   - Wiki API 获取知识库结构，返回 `node_token` 和 `obj_token`
   - Docx API 使用 `obj_token` 读写文档内容
   - 如果文档分享给了机器人，可以直接用 Docx API 访问
